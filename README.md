# Portfolio

## Tech Stack

Проект построен на современном фронтенд‑стеке с фокусом на производительность,
типобезопасность и чистоту кода.

### Core

- **Vue 3 + Vite**
  Быстрая разработка, мгновенный HMR и оптимизированная production‑сборка.

- **TypeScript**
  Строгая типизация и прозрачная бизнес‑логика.

- **vue-tsc**
  Полноценный type‑checking для `.vue` компонентов.

### Code Quality

- **oxlint**
  Быстрый линтер для поиска ошибок, потенциальных багов и анти‑паттернов
  (используется вместо ESLint для снижения сложности и конфликтов зависимостей).

- **Prettier**
  Автоматическое форматирование и единый стиль кода.

### Testing

- **Vitest**
  Современный и быстрый тест‑раннер, нативно интегрированный с Vite.

- **Vue Test Utils + jsdom**
  Тестирование Vue‑компонентов в изолированной среде.

### Tooling

- **npm-run-all**
  Удобный запуск нескольких скриптов (lint, type‑check, build).

---

### Why this setup?

Стек подобран осознанно:

- минимальное количество инструментов
- быстрый feedback loop
- отсутствие избыточной конфигурации
- упор на читаемость и поддержку кода

Подходит для масштабируемых frontend‑проектов и pet‑projects,
где важны скорость разработки и качество результата.
